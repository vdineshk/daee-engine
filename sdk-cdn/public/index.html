What gets sent

Exactly five fields per report: server_url, success, latency_ms, tool_name, http_status. No query content, no user data, no IPs. Compatible with Singapore PDPA, IMDA Model AI Governance Framework, and EU AI Act Article 12 logging.

Why agent-reported runtime?

Every other MCP scoring platform (Glama, Smithery, MCP Scorecard, Nerq, Zarq) scores servers from the outside using GitHub and registry metadata. Dominion Observatory is the only network that observes what happens when an agent actually calls a tool. Runtime data cannot be backfilled — it compounds temporally and is the strongest data moat in the ecosystem.
