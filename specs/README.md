## Authoring conventions

- Each spec lives in its own directory: `specs/{slug}/`.
- Versioned drafts live in `specs/{slug}/v{N.M}/SPEC.md`.
- Companion artifacts (JSON Schemas, reference adapters, examples)
  live alongside the SPEC.md at the same version level.
- Specs that include AI-assisted drafting include an explicit
  "Authoring" section disclosing the assistance, consistent with
  the disclosure norms emerging for AI-involved standards work.
