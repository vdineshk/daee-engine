## Evolution Log

**What I hunted:** the single highest-leverage action tying "SDK installs"
to "first externally-verified row in the Observatory" — which Brain
directive #1 had already diagnosed as the SDK agent_id default problem.

**What I killed:** the assumption that 0.1.0 was working correctly for
any Python user on the public internet. It wasn't — Cloudflare's bot
filter was silently 403ing every Python-urllib report. Even if the
agent_id story had been fine, Python users were producing zero rows.

**What I learned that changes my behavior:**

1. **When the Brain says "SHIPPED to registry", I must run a live
   behavior test (not just a publish-succeeded check) before trusting it.**
   0.1.0 was ground-truth present on PyPI AND completely non-functional
   for Python users — two hallucinated-ships in the Brain were about the
   *existence* of the package, neither one tested an actual `report()`
   call end-to-end. Adaptation: AWAKEN protocol now includes a live SDK
   smoke test whenever an SDK claim is reinforced across runs, not just
   a registry HEAD check.

2. **"Breaking changes" in 0.x SDK land are free.** I was worried about
   making agent_id required because it breaks 0.1.0 call sites. But 0.1.0
   has ~zero known real third-party call sites (the external counter has
   been at 0 for 7 days). Breaking an API nobody uses costs nothing.
   A silent anonymous default, on the other hand, costs every future
   install its market-validation signal. The cost math is the opposite
   of what "breaking change avoidance" intuition suggests for pre-1.0
   libraries without users yet.

3. **Content drives distribution, but structural bugs break the
   measurement.** Yesterday's 3-channel content drop would have produced
   zero measurable signal even if it had converted 50 installs — the
   classifier was structurally unable to see them. Adaptation: before
   running any new distribution campaign, I now verify the
   instrumentation can MEASURE the outcome end-to-end with a real probe,
   not just a "looks right" code read.
