Available Tools

MCP-compatible tools your AI agent can call directly.

calculate_cpf

CPF Contribution Calculator

Calculate employee and employer CPF contributions based on age, salary, citizenship status, and year. Returns OA/SA/MA allocation breakdowns.

get_cpf_rates

CPF Contribution Rates

Retrieve the full CPF contribution rate table by age group and citizenship status. Includes employee share, employer share, and total rates.

calculate_take_home

Take-Home Pay Calculator

Compute net take-home pay after CPF deductions. Factors in ordinary wages, additional wages, and annual wage ceiling.

get_cpf_ceilings

CPF Salary Ceilings

Get current and historical CPF ordinary wage ceiling, additional wage ceiling, and annual limit by year for payroll compliance.

Quick Start

Call the MCP endpoint directly with curl.

terminal 
curl -X POST https://sg-cpf-calculator-mcp.sgdata.workers.dev/mcp \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "method": "tools/call",
    "params": {
      "name": "calculate_cpf",
      "arguments": {
        "age": 30,
        "monthly_salary": 6000,
        "citizenship": "citizen"
      }
    }
  }'
response 
{
  "result": {
    "monthly_salary": 6000,
    "employee_contribution": 1200,
    "employer_contribution": 1020,
    "total_cpf": 2220,
    "take_home_pay": 4800,
    "allocation": {
      "ordinary_account": 1380,
      "special_account": 540,
      "medisave_account": 300
    }
  }
}
