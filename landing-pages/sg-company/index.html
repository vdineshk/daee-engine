Available Tools

MCP-compatible tools your AI agent can call directly.

lookup_company

Company Lookup by UEN

Search for Singapore-registered companies by UEN or company name. Returns registration details, address, SSIC codes, and incorporation date.

get_company_officers

Company Officers

Retrieve directors, shareholders, and company secretaries for a given UEN. Includes appointment dates and nationality information.

get_industry_stats

Industry Statistics

Get aggregate industry statistics by SSIC code including number of active companies, average paid-up capital, and incorporation trends.

check_company_status

Status Verification

Verify whether a company is live, struck off, dissolved, or under judicial management. Essential for due diligence and KYC workflows.

Quick Start

Call the MCP endpoint directly with curl.

terminal 
curl -X POST https://sg-company-lookup-mcp.sgdata.workers.dev/mcp \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -d '{
    "method": "tools/call",
    "params": {
      "name": "lookup_company",
      "arguments": {
        "query": "200301234A"
      }
    }
  }'
response 
{
  "result": {
    "uen": "200301234A",
    "company_name": "EXAMPLE PTE. LTD.",
    "status": "Live",
    "registration_date": "2003-01-15",
    "ssic_code": "62011",
    "ssic_description": "Software development",
    "address": "1 Raffles Place, Singapore 048616"
  }
}
